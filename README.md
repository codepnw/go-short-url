# Short URL
Generate short url using **golang, mongodb**
## Installation
```bash
git clone https://github.com/codepnw/go-short-url.git

cd go-short-url

## run
go run cmd/main.go

## build
go build -o short-url cmd/main.go
```

## Config

create **.env** as root path
```bash
DB_HOST=
API_VERSION=
PORT=
```
